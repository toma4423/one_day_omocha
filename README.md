# 🎁 今日のおもちゃ箱 (One Day Omocha)

Streamlit を使用した、日常生活やゲームで使えるミニツール・ミニゲーム集です。
ブラウザさえあれば、PC でもスマホでも、すぐに遊んだり記録したりできます。

## 🚀 特徴
- **マルチデバイス対応**: スマートフォンでも操作しやすい大きなボタンとレイアウト。
- **自動保存機能 (LocalStorage)**: ブラウザを閉じたりリロードしたりしても、自分のデータが保持されます。
- **日本時間 (JST) 対応**: 履歴の記録やファイル名はすべて日本時間に基づいています。
- **セーブ＆ロード**: 盤面の状態を JSON ファイルとして保存・復元できます（ビンゴ、カウントサポート）。
- **プライバシー配慮**: データは利用者のブラウザ内にのみ保存され、サーバーには送信されません。

## 📦 収録されているおもちゃ
### 🎲 ゲーム・エンタメ系
1. **サイコロ**: 最大100個のダイスを一度に振れます。出目履歴の保存に対応。
2. **双六メーカー**: 自由にマス目を作れるすごろく。自動コマ移動とループ機能付き。
3. **黒ひげ危機一発**: ハラハラドキドキの定番ゲーム。スロット番号表示で遊びやすく改善。
4. **マインスイーパー**: クラシックな地雷撤去ゲーム。
5. **チンチロリン**: 本格的な役判定とアニメーション。役の履歴保存が可能。

### 🔢 サポート・ツール系
1. **カウントサポート**: 複数の項目（X, Y, Z）に倍率をかけて計算できるカウンタ。
2. **カウントサポートビンゴ**: 最大 15×15 のマス目で複数の項目を同時カウント。JSONでの保存に対応。

## 🛠 開発者向け情報

### 技術スタック
- **Python 3.12+**
- **Streamlit**: Web UI フレームワーク
- **uv**: 高速な Python パッケージ・プロジェクト管理
- **pytest**: 自動テスト
- **streamlit-local-storage**: ブラウザへのデータ永続化

### セットアップと実行
このプロジェクトは `uv` を使用して管理されています。

```bash
# リポジトリのクローン
git clone https://github.com/toma4423/one_day_omocha.git
cd one_day_omocha

# 依存関係のインストールと実行
uv run streamlit run app.py
```

### テストの実行
```bash
uv run pytest
```

### プロジェクト構成
- `app.py`: メイン（ホーム）画面
- `pages/`: 各おもちゃの個別ページ
- `src/utils/`: 共通ロジック（ストレージ、サイコロ、時刻、スタイル等）
- `tests/`: 自動テストコード
